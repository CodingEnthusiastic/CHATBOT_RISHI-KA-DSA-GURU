```bash

```

### 3. Set Environment Variables

Create a `.env.local` file in the root directory:

```env

NEXT_PUBLIC_GEMINI_MODEL=gemini-1.5-flash
```

### 4. Run the Development Server

```bash
npm run dev
```

### 5. Build for Production

```bash
npm run build
npm start
```
